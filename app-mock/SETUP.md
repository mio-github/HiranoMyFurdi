# MyFurdi モックアプリ セットアップガイド

## クイックスタート

```bash
# 1. app-mockディレクトリに移動
cd "app-mock"

# 2. 依存関係をインストール
npm install

# 3. 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開くと、モックアプリが表示されます。

## 📱 表示される画面

### iPhone フレーム内に表示されるモック
- iPhoneの外観を模したフレーム
- ノッチ（切り欠き）の再現
- 実際のiPhone 14 Proサイズ（393px）

### 実装されている画面（5つ）

1. **ホーム画面** 📱
   - 時間帯別挨拶
   - 今日のチャレンジ
   - 来店サマリー
   - 混雑状況
   - 健康データ
   - クイックアクション

2. **レポート画面** 📊
   - 運動記録グラフ
   - 体組成データ
   - 来館履歴カレンダー

3. **入館証画面** 🎫
   - QRコード表示
   - 会員情報
   - 店舗選択

4. **リワード画面** 🏆
   - 日次チャレンジ
   - 週間・月間チャレンジ
   - バッジコレクション
   - 実績・ランク

5. **メニュー画面** ⚙️
   - プロフィール
   - 各種設定
   - サポート

## 🔧 カスタマイズ

### カラー変更
`tailwind.config.js` で色を変更できます：

```js
colors: {
  'furdi-pink': '#FF69B4',  // メインカラー
  'furdi-pink-light': '#FFE4E1',  // ライトカラー
}
```

### 画面追加
`src/components/` に新しいコンポーネントを追加し、
`App.tsx` でインポート・登録します。
