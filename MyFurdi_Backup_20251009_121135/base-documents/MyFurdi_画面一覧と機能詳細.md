## 3. 各画面の詳細仕様

---

### 2.1.1 スプラッシュ画面

**目的**: アプリ起動時のブランドイメージ表示・初期化処理

**表示要素**:
- FURDIロゴ（中央配置）
- ブランドカラー（ピンク）の背景グラデーション
- ローディングインジケータ

**遷移先**:
- ログイン済み → ホーム画面
- 未ログイン → ログイン画面

---

### 2.1.2 ログイン画面

**目的**: 会員認証

**表示要素**:
- アプリロゴ
- キャッチコピー: "あなたのために 最高のあなたへ"
- メールアドレス入力フィールド
- パスワード入力フィールド
- ログインボタン（プライマリ）
- パスワードを忘れた方へ（リンク）
- 新規会員登録ボタン（セカンダリ）

**機能**:
- メール・パスワード認証
- ソーシャルログイン（Google/Apple）※将来対応
- エラーハンドリング（認証失敗時）

---

### 2.1.3 初回チュートリアル

**目的**: 初回ユーザーへの使い方ガイド

**表示要素**:
- スワイプ可能なカルーセル（3-5ページ）
  1. アプリの目的
  2. 主要機能の紹介（混雑確認・記録・リワード）
  3. プッシュ通知の許可リクエスト
- スキップボタン
- 次へ/始めるボタン

---

### 2.2.1 ホーム画面

**目的**: ユーザーが最も頻繁に確認する情報を集約し、即時フィードバックとエンゲージメントを最大化するダッシュボード

**設計思想**:
- **即時性**: 開いた瞬間に「今知りたい情報」が目に入る
- **達成感**: 小さな成功体験を視覚的に強調
- **行動誘導**: 次のアクションを自然に促す配置

---

**レイアウト構成**（優先度順・上から順に）:

#### ① ヘッダー
- **時間帯別挨拶**:
  - 朝（6-11時）: "おはようございます、〇〇さん！"
  - 昼（12-17時）: "こんにちは、〇〇さん！"
  - 夜（18-5時）: "こんばんは、〇〇さん！"
- **通知アイコン**（バッジ付き、未読数表示）
- **設定アイコン**

#### ② 今日のチャレンジカード（最優先表示）
**"低いエンゲージメント入口"の実装**

**未達成時**:
```
┌─────────────────────────────┐
│ 🎯 今日のチャレンジ           │
│                              │
│ ☑️ 体重を記録する            │
│   報酬: デイリーバッジ 🏅     │
│                              │
│ [1タップで記録する] ボタン    │
└─────────────────────────────┘
```

**達成済み時**（祝福演出付き）:
```
┌─────────────────────────────┐
│ ✨ 完了しました！             │
│                              │
│ デイリーバッジを獲得 🏅       │
│ 3日連続達成中！               │
│                              │
│ 明日のチャレンジでまた会いましょう │
└─────────────────────────────┘
```

**機能**:
- タスクをタップ → 該当画面へディープリンク
- 達成時: 紙吹雪アニメーション + サウンド（オプション）
- 連続達成日数（ストリーク）を強調表示

---

#### ③ 来店サマリカード（目標達成を視覚化）
**データ駆動のモチベーション維持**

```
┌─────────────────────────────┐
│ 🔥 5日連続来店中！           │
│                              │
│ 今週の来店 ████░░ 3/4回      │
│ "あと1回で目標達成です！"     │
│                              │
│ 前回: 昨日 18:30             │
│ 次回おすすめ: 明日 18:00頃   │
│ （この時間帯は空いています） │
└─────────────────────────────┘
```

**表示要素**:
- **連続来店日数**: 大きく強調（炎アイコン付き）
- **週間目標進捗**: プログレスバー + 数値
- **励ましメッセージ**: 目標に応じた動的メッセージ
  - 達成間近: "あと1回で目標達成！"
  - 達成済み: "目標達成！素晴らしいです✨"
  - 遅れ気味: "次回の来店をお待ちしています"
- **前回来店**: 日時表示
- **AI推奨時間**: 行動パターン + 混雑状況から最適時間を提案

---

#### ④ 混雑状況クイックビュー（リアルタイム情報）
**情報の即時性を提供**

```
┌─────────────────────────────┐
│ 📍 〇〇店（お気に入り）       │
│                              │
│ 🟢 かなり空いています        │
│ 現在の利用率: 25%            │
│                              │
│ [詳しく見る] [今から行く]    │
└─────────────────────────────┘
```

**表示要素**:
- **店舗名** + お気に入りマーク
- **混雑度**: 色 + アイコン + テキスト
  - 🟢 かなり空いています（0-30%）
  - 🟡 やや混雑（31-60%）
  - 🟠 混雑（61-80%）
  - 🔴 かなり混雑（81-100%）
- **利用率**: パーセント表示
- **アクションボタン**:
  - 「詳しく見る」: 混雑状況詳細画面へ
  - 「今から行く」: 経路案内アプリ起動（Googleマップ等）

**更新頻度**: 5分ごと（自動）、プルダウンで即時更新

---

#### ⑤ 健康データサマリー（即時フィードバック）
**タブ切替式のコンパクト表示**

**タブ**: 体重 / 体脂肪 / 歩数（スワイプで切替）

**体重タブ例**:
```
┌─────────────────────────────┐
│ ⚖️ 体重                      │
│                              │
│ 54.2 kg  ↓ -0.3kg           │
│ （前回比）                   │
│                              │
│ ▂▃▅▃▄▅▆ （7日間のグラフ）   │
│                              │
│ "順調に減っています✨"        │
│ [記録する]                   │
└─────────────────────────────┘
```

**表示要素**:
- **最新値**: 大きく強調表示
- **前回比**: ↑↓アイコン + 差分（色分け）
  - 改善: グリーン
  - 横ばい: グレー
  - 悪化: オレンジ（ネガティブな表現は避ける）
- **ミニグラフ**: 直近7日間のトレンド
- **励ましメッセージ**: データに応じた前向きなコメント
- **記録ボタン**: 1タップで入力モーダル

**歩数タブ（将来対応）**:
- ヘルスケア連携（iOS）/ Google Fit連携（Android）
- 本日の歩数、目標達成率、消費カロリー

---

#### ⑥ クイックアクション（アイコングリッド）
**頻繁に使う機能への最短アクセス**

```
┌─────────────────────────────┐
│ [🎬 動画] [📝 記録] [📊 データ] │
│ [📰 お知らせ] [❓ FAQ] [🧬 遺伝子] │

```

**表示要素**:
- 2列×3行のアイコングリッド
- アイコン + ラベル
- タップで各機能へ遷移

**推奨機能**:
- 動画トレーニング
- 運動記録
- レポート
- お知らせ
- FAQ
- 遺伝子テスト結果

---

#### ⑦ 週次サマリーカード（折りたたみ可能）
**成長の実感を提供**

```
┌─────────────────────────────┐
│ 📈 今週の成果                │
│                              │
│ トレーニング時間: 120分      │
│ （先週比 +20分 ⬆️）          │
│                              │
│ 消費カロリー: 850 kcal       │
│ 来店回数: 3回                │
│                              │
│ ▂▃▅▃▄▅▆ （週間グラフ）      │
│                              │
│ [詳しく見る]                 │
└─────────────────────────────┘
```

**表示要素**:
- トレーニング時間合計
- 消費カロリー合計
- 来店回数
- **先週比較**: 増減を表示（ポジティブ強調）
- ミニグラフ
- 詳細へのリンク

**デフォルト状態**: 折りたたみ（タップで展開）

---

#### ⑧ お知らせバナー（下部）
**重要情報の通知**

- 重要なお知らせのみ表示（1-2件）
- カルーセル形式
- タップで詳細へ遷移
- 既読/未読管理

---

**インタラクション設計**:

1. **プルダウンリフレッシュ**:
   - 全データを再取得
   - 混雑状況を即時更新
   - 新しいチャレンジをチェック

2. **カードタップ**:
   - 各カード全体がタップ可能
   - リップルエフェクトでフィードバック
   - 詳細画面へスムーズ遷移

3. **チャレンジ達成時**:
   - 紙吹雪アニメーション（3秒）
   - サウンド再生（設定でON/OFF）
   - バッジアイコンをカードに表示
   - 「リワード画面で確認」への導線

4. **データ更新時**:
   - パルスアニメーション
   - 新しい値を一時的にハイライト

**パフォーマンス**:
- 初回表示: 1.5秒以内（スケルトンスクリーン表示）
- リフレッシュ: 1秒以内
- カード遷移: 300ms以内

### 2.2.2 レポート画面

**目的**: 運動記録・体組成データの詳細閲覧・分析

**タブ構成**:
1. **運動記録**
2. **体組成**
3. **来館履歴**

---

#### タブ1: 運動記録

**期間フィルタ**:
- 日 / 週 / 月 / 年

**表示要素**:
- **トレーニング時間グラフ**（棒グラフ）
- **運動種目別内訳**（円グラフ）
- **履歴リスト**:
  - 日付
  - 運動種目（例: AIマシン、自宅トレーニング動画）
  - 時間・強度
  - メモ

**データソース**:
- PIXFORMANCE（AIマシン）
- 手動入力（動画トレーニング完了後）

**機能**:
- 記録の追加・編集・削除
- CSVエクスポート（将来対応）

---

#### タブ2: 体組成

**期間フィルタ**:
- 日 / 週 / 月 / 年

**表示要素**:
- **体重推移グラフ**（折れ線グラフ）
- **体脂肪率推移グラフ**（折れ線グラフ）
- **その他指標**:
  - BMI
  - 筋肉量
  - 内臓脂肪レベル
- **履歴リスト**:
  - 測定日時
  - 各指標の数値
  - 前回比

**データソース**:
- TANITA測定機器（自動連携）
- 手動入力

**機能**:
- 目標設定（目標体重・体脂肪率）
- トレンド分析（週平均・月平均）
- 記録の手動追加

---

#### タブ3: 来館履歴

**期間フィルタ**:
- 週 / 月 / 年

**表示要素**:
- **カレンダービュー**:
  - 来店日にマーク表示
  - 連続来店日数のハイライト
- **履歴リスト**:
  - 日付
  - 入館時刻・退館時刻
  - 滞在時間
  - 店舗名

**データソース**:
- SECOM入退館システム（自動連携）

**機能**:
- カレンダー/リスト表示切替
- 来店頻度分析（週何回・最も混雑している時間帯）

### 2.2.4 リワード画面

**目的**: ゲーミフィケーション要素でユーザーの継続的なエンゲージメントを促進

**設計思想**:
- **3層構造**: 日次 → 週次・月次 → 長期実績の段階的チャレンジ
- **視覚的報酬**: バッジコレクションで達成感を演出
- **進捗の可視化**: プログレスバーで「あと少し」を実感
- **サプライズ要素**: 予期せぬ報酬で楽しさを追加

**タブ構成**:
1. **今日のチャレンジ**（日次）
2. **週間・月間チャレンジ**（中期）
3. **マイバッジ**（コレクション）
4. **実績**（長期）

---

#### タブ1: 今日のチャレンジ（Layer 1: 低いハードル）

**目的**: 毎日アプリを開く習慣を作る

**レイアウト**:
```
┌─────────────────────────────────┐
│ 🎯 今日のチャレンジ              │
│ 全体進捗: ████░░ 3/5 達成       │
│                                 │
│ ✅ アプリを開く                  │
│    報酬: デイリースタンプ 🎫     │
│                                 │
│ ✅ 体重を記録する                │
│    報酬: ヘルスバッジ 🏅         │
│                                 │
│ ☐ 混雑状況を確認する            │
│    報酬: スマートバッジ 🔍       │
│    [今すぐ確認] ←ディープリンク │
│                                 │
│ ☐ 動画を1本視聴する              │
│    報酬: トレーナーバッジ 🎬     │
│    [動画を見る]                  │
│                                 │
│ ☐ 来店する                      │
│    報酬: フィットネスバッジ 💪   │
│                                 │
├─────────────────────────────────┤
│ 🎁 全達成ボーナス                │
│ 5つすべて達成で特別バッジ獲得！  │
└─────────────────────────────────┘
```

**表示要素**:
- **全体進捗バー**: 達成数 / 全タスク数
- **タスクリスト**（3-5個）:
  - ✅ 達成済み（グリーン背景）
  - ☐ 未達成（デフォルト）
  - 🔒 条件未達成（グレーアウト）
- **各タスク**:
  - タスク名
  - 報酬アイコン + 名前
  - アクションボタン（未達成時のみ）
- **全達成ボーナス**: 特別な報酬を提示

**タスクカテゴリ**:
1. **エントリータスク**（確実に達成可能）:
   - アプリを開く
   - ホーム画面を見る

2. **記録タスク**（低ハードル）:
   - 体重を記録する
   - 運動を記録する
   - 混雑状況を確認する

3. **行動タスク**（中ハードル）:
   - 来店する
   - 動画を1本視聴する
   - AIマシンを使う

4. **連続タスク**（習慣化）:
   - 3日連続記録
   - 7日連続来店

**ダイナミックタスク生成**:
- ユーザーの行動パターンに応じてタスクを自動調整
- 例: 最近来店していない → 「来店する」タスクを追加
- 例: 体重記録が途切れている → 「体重を記録する」を優先表示

---

#### タブ2: 週間・月間チャレンジ（Layer 2: 中程度のハードル）

**目的**: 継続的な努力を促し、習慣化を支援

**レイアウト**:
```
┌─────────────────────────────────┐
│ 📅 週間チャレンジ（今週）        │
│                                 │
│ 🏃 週3回来店チャレンジ           │
│ 進捗: ████░░░ 2/3 回            │
│ 報酬: ウィークリーチャンピオン 🏆 │
│ 残り期間: あと3日                │
│                                 │
│ 🎬 動画3本視聴チャレンジ         │
│ 進捗: ██████ 3/3 本 ✅          │
│ 報酬: ビデオマスター 📺          │
│ [報酬を受け取る]                │
│                                 │
├─────────────────────────────────┤
│ 📆 月間チャレンジ（今月）        │
│                                 │
│ 💪 累計10時間トレーニング        │
│ 進捗: ████████░░ 8.5/10 時間    │
│ 報酬: マンスリーアスリート 🥇    │
│ あと1.5時間で達成！              │
│                                 │
│ 📉 -1kg達成チャレンジ            │
│ 進捗: ████░░░░░░ 0.4/1.0 kg     │
│ 報酬: ダイエットサクセス 🎯      │
│ 順調です、その調子！             │
└─────────────────────────────────┘
```

**表示要素**:
- **週間チャレンジセクション**:
  - 2-3個のチャレンジ
  - 各チャレンジカード:
    - タイトル + アイコン
    - プログレスバー + 数値
    - 報酬プレビュー
    - 残り期間
    - 達成時: 「報酬を受け取る」ボタン
    - 未達成: 励ましメッセージ

- **月間チャレンジセクション**:
  - 2-3個のチャレンジ
  - 長期目標を設定
  - 週間より難易度高め
  - より豪華な報酬

**チャレンジ例**:

**週間**:
- 週3回来店
- 動画3本視聴
- 毎日体重記録（7日連続）
- 累計2時間トレーニング

**月間**:
- 累計10時間トレーニング
- 体重-1kg達成
- 15回来店
- すべての動画カテゴリを視聴

**期間管理**:
- 週間: 月曜日0時 〜 日曜日23:59
- 月間: 月初1日0時 〜 月末23:59
- 期間終了時: 自動リセット + 次のチャレンジ生成

## 4. 画面遷移図

### 4.1 メインナビゲーション（タブバー）

**5つのタブで主要画面を切り替え**:
```
ホーム画面 ←→ レポート画面 ←→ 入館証画面 ←→ リワード画面 ←→ メニュー画面
```

### 4.2 主要な画面遷移フロー

#### ログイン〜ホーム
```
ログイン画面 → 初回チュートリアル（初回のみ） → ホーム画面
```

#### ホーム画面からの遷移
```
ホーム画面
  ├─→ 混雑確認詳細 → 店舗選択/予約
  ├─→ 体重記録モーダル
  ├─→ 動画トレーニング一覧 → 動画再生
  └─→ レポート画面（健康データ詳細）
```

#### レポート画面の詳細遷移
```
レポート画面
  ├─→ 運動記録タブ → 記録詳細
  ├─→ 体組成タブ → 推移グラフ
  └─→ 来館履歴タブ → 履歴詳細
```

#### メニュー画面からの遷移
```
メニュー画面
  ├─→ 動画トレーニング一覧
  ├─→ お知らせ一覧 → お知らせ詳細
  ├─→ FAQ・コラム
  ├─→ DNA検査結果画面（NEW）
  ├─→ 通知設定
  └─→ 設定画面
```

#### DNA検査結果画面（新規追加）
```
メニュー画面 → DNA検査結果画面
                 ├─→ 総合評価タブ
                 ├─→ 筋肉タイプタブ
                 ├─→ 代謝タブ
                 ├─→ 怪我リスクタブ
                 └─→ 詳細データタブ → PDFダウンロード
```
