# MyFurdi SVGワイヤーフレーム デザインポリシー

## 目的
本ドキュメントは、MyFurdiアプリのSVGワイヤーフレーム作成における統一されたデザインポリシーを定めるものです。

## 基本仕様

### サイズとフォーマット
- **画面サイズ**: 393×852px (iPhone 14 Pro基準)
- **画面遷移図**: 1400×1200px (全体俯瞰用)
- **ファイル形式**: SVG (Scalable Vector Graphics)

### カラーパレット

#### ブランドカラー
- **FURDI Pink**: `#FF69B4` (メインブランドカラー)
- **FURDI Deep Pink**: `#FF1493` (グラデーション用)

#### システムカラー
- **背景色**: `#F5F5F5` (画面背景)
- **カード背景**: `#FFFFFF` (白)
- **テキスト**: `#1C1C1E` (濃いグレー、メインテキスト)
- **サブテキスト**: `#3A3A3C` (中間グレー、補助テキスト)
- **ボーダー**: `#E5E5E7` (薄いグレー、境界線)

#### 機能別カラー
- **成功・接続済み**: `#4CAF50` (緑)
- **警告・要対応**: `#FF9800` (オレンジ)
- **エラー**: `#FF0000` (赤)
- **情報**: `#2196F3` (青)
- **Phase2予算内**: `#4CAF50` (緑)
- **Phase2追加予算**: `#FF9800` (オレンジ)

### タイポグラフィ

#### フォント
- **フォントファミリー**: Arial (SVG標準フォント)
- **太字**: `font-weight: bold`

#### フォントサイズ
- **大見出し**: 24px (画面遷移図タイトル)
- **中見出し**: 18px (画面名)
- **小見出し**: 16px (セクション名)
- **通常テキスト**: 13px (機能名)
- **本文**: 11-12px (説明文)
- **キャプション**: 9-10px (補足情報)
- **バッジ**: 8-9px (Phase表記など)

### レイアウト

#### 画面ワイヤーフレーム（393×852px）
```
┌─────────────────────────────┐
│ ステータスバー (44px)        │
├─────────────────────────────┤
│ ヘッダー (60px)              │
├─────────────────────────────┤
│                             │
│ コンテンツエリア            │
│ (可変高さ)                  │
│                             │
├─────────────────────────────┤
│ タブバー (60px)              │
└─────────────────────────────┘
```

#### 余白・間隔
- **画面端マージン**: 16px
- **カード間隔**: 8-16px
- **カード角丸**: 12px (大), 8px (小)
- **ボタン角丸**: 20-25px (丸みのある)
- **テキスト行間**: 1.2-1.5em

## 画面遷移図の特別ルール

### フェーズ表示

#### Phase 1（予算内）
- **枠線**: `#FF69B4` (FURDI Pink)、実線、幅3-4px
- **背景**: `#FFFFFF`
- **バッジ**: 不要（Phase1はデフォルト）

#### Phase 2（予算内）
- **枠線**: `#4CAF50` (緑)、点線（8,4）、幅3px
- **背景**: `#E8F5E9` (薄緑)
- **バッジ**: 緑背景に白文字「Phase2」
- **矢印**: 緑点線

#### Phase 2（追加予算）
- **枠線**: `#FF9800` (オレンジ)、点線（8,4）、幅3px
- **背景**: `#FFF3E0` (薄オレンジ)
- **バッジ**: オレンジ背景に白文字「Phase2」
- **矢印**: オレンジ点線

### 矢印スタイル

#### 認証フロー
- **色**: `#2196F3` (青)
- **幅**: 4px
- **スタイル**: 実線
- **マーカー**: `#arrowBlue`

#### タブバー遷移
- **色**: `#FF69B4` (FURDI Pink)
- **幅**: 2px
- **スタイル**: 実線
- **マーカー**: `#arrowPink`

#### アクション遷移
- **色**: `#4CAF50` (緑)
- **幅**: 2px
- **スタイル**: 点線（5,5）
- **マーカー**: `#arrowGreen`

#### Phase2予算内遷移
- **色**: `#4CAF50` (緑)
- **幅**: 2px
- **スタイル**: 点線（8,4）
- **マーカー**: `#arrowGreen`

#### Phase2追加予算遷移
- **色**: `#FF9800` (オレンジ)
- **幅**: 2px
- **スタイル**: 点線（8,4）
- **マーカー**: `#arrowOrange`

### 凡例の配置
- **位置**: 画面上部（y=90付近）
- **要素間隔**: 150-200px
- **テキストサイズ**: 10px
- **見出しサイズ**: 12px（太字）

## テキスト表記ルール

### 冗長表現の回避
❌ 避けるべき表現:
- 「Phase2予算内」「Phase2追加予算」（キャプション内）
- 「(予算内)」「(追加予算)」（矢印ラベル内）

✅ 推奨表現:
- バッジ: 「Phase2」のみ
- 矢印ラベル: 機能名のみ（例: 「DNA結果」「AIに相談」）
- **色で予算区分を表現**（緑=予算内、オレンジ=追加予算）

### 画面名の省略ルール
- 「DNA検査結果画面」→「DNA結果」
- 「DNA-AIチャット画面」→「AIチャット」
- 長い名称は12文字以内に省略

### アノテーション
画面下部に以下の形式で記載:
```svg
<text x="10" y="870" font-family="Arial" font-size="10" fill="#666">
  画面名: ○○画面 | 主要機能: ○○、○○、○○
</text>
```

## アイコンの使用

### 絵文字アイコン
- サイズ: 16-24px
- 配置: テキストの前、中央揃え
- 例: 🏠🔐📊🎫🏆⚙️🧬🤖

### 機能アイコン（丸背景）
```svg
<circle cx="15" cy="60" r="5" fill="#FFE4E1"/>
```
- **Phase1機能**: `#FFE4E1` (薄ピンク)
- **Phase2予算内**: `#E8F5E9` (薄緑)
- **Phase2追加予算**: `#FFF3E0` (薄オレンジ)

## アクセシビリティ

### 色覚対応
- **色のみに依存しない**: テキストとアイコンの併用
- **コントラスト比**: WCAG AA準拠（4.5:1以上）
- **パターン併用**: 点線/実線で区別

### 読みやすさ
- **最小フォントサイズ**: 8px
- **行間**: 適切な余白確保
- **情報の階層化**: 見出し→本文→注釈の順
